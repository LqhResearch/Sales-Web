## Ngôn ngữ lập trình và công cụ
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- Bootstrap
- jQuery

## Tính năng của website
- Tài khoản

  - Đăng ký
- Nhân viên giao hàng
  - Xem danh sách
  - Xuất danh sách excel
  - Thêm (bàn phím, excel)
  - Sửa
  - Xoá
  - Tìm kiếm (Tên nhân viên)
- Sản phẩm
  - Xem danh sách
  - Thêm
  - Sửa
  - Xoá
  - Tìm kiếm (Tên sản phẩm)
